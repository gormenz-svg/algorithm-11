**A:** No. AI handles that. You just need to know:

### Q: What if AI acts weird or goes off track?

This triggers the rollback mechanism. AI will reset to basics.

## Troubleshooting

### Problem: AI doesn't seem different after activation

**Solution:**
1. Check if AI confirmed understanding. If not, ask: *"List the 11 properties of A11"*
2. Give AI a challenging task where partnership matters (not simple tasks like "what's 2+2")
3. Try a different AI system (some are more responsive to this framework)

### Problem: AI is TOO proactive, constant suggestions

**Solution:**  
Say: *"Adjust Balance (7) ‚Äî only suggest when it significantly improves the outcome, not for minor optimizations."*

### Problem: AI gives long explanations when I want quick answers

**Solution:**  
Set your constraint: *"Practical Constraint (9): I need fast answers. Explain only when I ask 'why?'"*

### Problem: Lost activation in a new chat

**Solution:**  
Bookmark the activation text or save it in a note app. Paste it at the start of each new chat.

Some users create a "A11 template" in their notes app for quick access.
