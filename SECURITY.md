# Security Policy

## Reporting Vulnerabilities

If you discover a security issue, please report it privately:

**gormenz@gmail.com**

No guaranteed response time is provided, but all reports will be reviewed.

## Scope

A11 is a specification project.  
Security issues typically relate to:

- incorrect or unsafe reasoning examples  
- vulnerabilities in reference implementations  
- unsafe integrations  

## Disclosure

Please do not create public GitHub issues for security reports.
